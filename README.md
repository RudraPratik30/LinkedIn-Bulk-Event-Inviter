# LinkedIn Bulk Event Inviter Pro 🚀

A lightweight Chrome Extension (Manifest V3) built to automate the tedious process of selecting connections for LinkedIn Events. Designed with human-mimetic delays to stay under the radar and a real-time progress tracker.

## ✨ Features
- **Bulk Selection:** Enter a target number (e.g., 100) and let it work.
- **Auto-Scroll:** Automatically handles LinkedIn's lazy-loading/virtual DOM.
- **Human Jitter:** Randomized delays (300ms-600ms) to mimic real user behavior.
- **Progress Tracking:** Real-time counter and progress bar in the popup.
- **Safety First:** Includes a "Stop" button to kill the script instantly.

## 🛠 Installation (Developer Mode)
1. Download the ZIP of this repository and Unzip it.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer Mode** (top right).
4. Click **Load unpacked** and select the project folder.
5. Pin the extension to your toolbar.

## 📖 How to Use
1. Navigate to your [LinkedIn Event page](https://www.linkedin.com/events/7444469892541702144/).
2. Click the **"Share" > "Invite"** button to open the connection selection modal.
3. (Optional) Apply any LinkedIn filters (Location, Company, etc.).
4. Open the extension popup, enter your target count, and hit **Start Selecting**.
5. Once finished, click the "Invite" button on LinkedIn manually.
